void jobs::internal::fiber::convert_thread_to_fiber ( fiber &  result)
static

Converts the current thread to a fiber.

On some operating systems (notably windows) fibers can only be run by other fibers. so any worker threads dealing with fibers need to first be converted. Which is done using this function.
