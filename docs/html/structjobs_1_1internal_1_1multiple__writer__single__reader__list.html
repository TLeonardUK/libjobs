Detailed Description

template<typename data_type>
struct jobs::internal::multiple_writer_single_reader_list< data_type >

Thread-safe list that supports having multiple writers but a single reader.

This is primary used for things like wait-queues where multiple threads will want to add things to it, but only a single-thread would want to read (and block writes during the read).
