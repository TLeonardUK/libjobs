Detailed Description

template<typename data_type, int capacity>
struct jobs::internal::fixed_queue< data_type, capacity >

Fixed size, statically-allocated, thread-unsafe queue.

This is implemented internally as a ring buffer.

Template Parameters
data_typeType of value held in queue.
capacityMaximum number of values queue can hold.
